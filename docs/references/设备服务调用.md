# 设备服务调用

最近更新时间：2026-01-20 10:26:06

 

 

 

## 接口功能

根据产品ID和设备名，下发物模型服务调用命令到设备，设备返回服务调用的输出结果到接口

## 备注

仅支持OneJson设备，需要设备在线

## 接口地址

https://iot-api.heclouds.com/thingmodel/call-service

## API说明

请求方式：POST

**http body 请求参数：**

| 参数        | 类型   | 是否必选 | 描述                                     |
| :---------- | :----- | :------- | :--------------------------------------- |
| product_id  | string | 是       | 产品ID，平台生成唯一ID                   |
| device_name | string | 是       | 设备名称                                 |
| identifier  | string | 是       | 服务功能点标识                           |
| params      | object | 是       | 输入参数的键值对，输入参数的唯一标识做键 |

**返回数据**

| 参数名称   | 类型   | 描述                            |
| :--------- | :----- | :------------------------------ |
| code       | int    | 调用成功或失败时，返回的code码  |
| msg        | string | 调用成功或失败时，返回的msg信息 |
| request_id | string | 调用API生成的请求标识           |
| data       | -      | 调用成功时，返回的业务数据      |

## 示例

**请求示例**

```
POST 
http(s)://iot-api.heclouds.com/thingmodel/call-service
Content-type: application/json
{
  "product_id":"B7EEW578EbRg5Y4K",
  "device_name":"device3",
  "identifier": "light",
  "params":{
    "Power1":"1",
    "WF1":"2"
  }
}
```

**响应示例**

```
{
    "data":{
      "result1":"1",
      "result2":"2"
    },
    "request_id": "a25087f46df04b69b29e90ef0acfd115", 
    "msg": "succ",
    "code":  0
}
```
