统计支出信息

# 接口说明

统计支出信息

调用频率：5秒内只能调用一次

接口地址：

    POST https://stat.zsgai.com/big_analyse_pay_out_count

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

| 字段 | 数据类型 | 必填 | 默认值 | 说明 |
|:-----|----------|------|--------|------|
| 无   |          |      |        |      |

平台返回：

``` json
{
  "code": 0, // 0成功 其它失败
  "data": {
    "payOutCount": 9,        // 提现次数
    "payOutMoney": 5.34,     // 提现申请金额
    "pushSuccessMoney": 4.34,// 提现成功金额
    "refundUserMoney": 0     // 退款金额
  },
  "msg": "查询成功",
  "serverName": "stat.zsgai.com",
  "request_user_id": 9485,
  "request_refresh_code": null,
  "responseTime": 6
}
```
