统计投递信息

# 接口说明

统计投递信息

调用频率：5秒内只能调用一次

接口地址：

    POST https://stat.zsgai.com/big_analyse_delivery_count

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

| 字段 | 数据类型 | 必填 | 默认值 | 说明 |
|:-----|----------|------|--------|------|
| 无   |          |      |        |      |

平台返回：

``` json
{
  "code": 0, // 0成功 其它失败
  "data": {
    "deliveryCount": 11978,     // 本月投递次数
    "deliveryScore": 0,         // 本月投递积分
    "deliveryWeight": 18189.41, // 本月投递重量
    "deliveryMoney": 1832.24,   // 本月投递金额
    "minusyMoney": 0            // 本月投递审核异常金额
  },
  "msg": "查询成功",
  "serverName": "stat.zsgai.com",
  "request_user_id": 9485,
  "request_refresh_code": null,
  "responseTime": 86
}
```
