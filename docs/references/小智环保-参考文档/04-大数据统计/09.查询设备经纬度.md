查询设备经纬度

# 接口说明

查询设备经纬度,用于在地图上显示设备的坐标

调用频率：5秒内只能调用一次

接口地址：

    POST https://stat.zsgai.com/big_analyse_map_device

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

| 字段 | 数据类型 | 必填 | 默认值 | 说明 |
|:-----|----------|------|--------|------|
| 无   |          |      |        |      |
