统计清运信息

# 接口说明

统计清运信息

调用频率：5秒内只能调用一次

接口地址：

    POST https://stat.zsgai.com/big_analyse_clean_count

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

| 字段 | 数据类型 | 必填 | 默认值 | 说明 |
|:-----|----------|------|--------|------|
| 无   |          |      |        |      |

平台返回：

``` json
{
  "code": 0, // 0成功 其它失败
  "data": {
    "cleanCount": 64,          // 本月清运次数
    "totalWeights": 8292.13,   // 本月清运重量
    "cleanWeights": 8292.13,   // 本月清运重量  
    "storageWeights": 0,       // 本月清运入库重量
    "minusyWeights": 0         // 本月清运异常重量

  },
  "msg": "查询成功",
  "serverName": "stat.zsgai.com",
  "request_user_id": 9485,
  "request_refresh_code": null,
  "responseTime": 112
}
```
