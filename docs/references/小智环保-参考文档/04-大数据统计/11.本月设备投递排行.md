# 接口说明

本月设备投递排行

调用频率：5秒内只能调用一次，每1个小时更新一次排名

接口地址：

    POST https://stat.zsgai.com/big_analyse_device_delivery_sort

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

以下字段，用到哪个就携带哪个。

| 字段       | 数据类型 | 必填 | 默认值 说明       |
|:-----------|:---------|:-----|:------------------|
| page_size  | int      | 否   | 默认5条 最大50条  |
| startYear  | int      | 否   | 默认当前年份 2025 |
| startMonth | int      | 否   | 默认当前月份 2    |

``` json
    {

  ],

}
```
