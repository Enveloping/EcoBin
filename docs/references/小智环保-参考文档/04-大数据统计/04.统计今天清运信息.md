统计今天清运信息

# 接口说明

统计今天清运信息

调用频率：5秒内只能调用一次

接口地址：

    POST https://stat.zsgai.com/big_analyse_clean_count_today

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

| 字段 | 数据类型 | 必填 | 默认值 | 说明 |
|:-----|----------|------|--------|------|
| 无   |          |      |        |      |

平台返回：

``` json
{
  "code": 0, // 0成功 其它失败
  "data": {
    "todayCleanCount": 4,        // 今天清运次数
    "todayCleanWeightCount": 10  // 今天清运重量
  },
  "msg": "查询成功",
  "serverName": "stat.zsgai.com",
  "request_user_id": 9485,
  "request_refresh_code": null,
  "responseTime": 7
}
```
