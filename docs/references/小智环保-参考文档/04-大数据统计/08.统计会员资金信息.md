统计会员资金信息

# 接口说明

统计会员资金信息

调用频率：5秒内只能调用一次

接口地址：

    POST https://stat.zsgai.com/big_analyse_member_money_count

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

| 字段 | 数据类型 | 必填 | 默认值 | 说明 |
|:-----|----------|------|--------|------|
| 无   |          |      |        |      |

平台返回：

``` json
{
  "code": 0, // 0成功 其它失败
  "data": {
    "memberCount": 23,         // 会员数量 
    "memberMoney": 3799.51,    // 会员余额
    "memberScore": 0,          // 会员积分
    "memberPlanMoney": 0.9     // 会员预计金额 (在审核成功后会增加到会员余额中，审核为违规则消除此金额) 
  },
  "msg": "查询成功",
  "serverName": "stat.zsgai.com",
  "request_user_id": 9485,
  "request_refresh_code": null,
  "responseTime": 7
}
```
