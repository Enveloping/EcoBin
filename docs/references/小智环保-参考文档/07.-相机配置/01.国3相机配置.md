# 小智国标监控配置文档

小智国标为金牛星自主研发的监控平台

## 1、测试地址

<https://sip.xzhb.com（http://223.68.0.146:9093/>）

## 2、相机配置

通常出厂前会配置好，但对于特殊场景仍需要在现场配置，以下提供配置方法：\
相机配置需要访问相机的控制页面，与回收箱在同一局域网，需要到设备前配置，\
若提供了配置工具的相机，也可通过配置工具搜索配置。

### 2.1 相机中配置国标参数

SIP服务器域：3213110000\
SIP服务器编号：32131100002120000012\
SIP服务器密码：12345678\
SIP服务器IP(*)：223.68.0.146\
SIP服务器端口(*)：5060\
传输协议(*)：TCP (莹石国标使用UDP)\
国标协议版本(*)：GB28181-2016

示例： 设备国标编号(*) ：32131100001320006762\
通道编号(*)：32131100001320006762 (必须与设备国标编号相同)

### 2.2 视音频配置

国标不支持远程降级，请将分辨率、码率调整到最低以提高拍照效率和节省流量。\
若因较高的的分辨率和码流占用较高的带宽时，后台会不定期进行屏蔽，由此可能导致不能拍照和预览。

参考配置：

#### 2.2.1 主码流

        视频编码：H.265
        码率上限 1280*720  
        视频帧率：25
        I桢间隔：60
        码流平滑：30

#### 2.2.2 子码流

        视频编码：H.265
        码率上限 704*576
        视频帧率：25
        I桢间隔：60
        码流平滑：30

### 2.3 回放设置

非标相机不支持远程开启录相，\
要在本地格式化sd卡，并开启全天24小时录相。

## 3、测试

访问 <https://sip.xzhb.com> ，选择对应已上线的设备，确保可以预览相机，\
当回放配置后，过一会(5-30分钟)观察是否有录相产生，并可以通过回放查看录相。

## 4、绑定到回收箱平台

在设备配置页面\>设备信息中编辑：\
相机厂商选择：\[自主研发监控平台\]\
填写国标编码，示例 32131100001320006762\
修改后，将配置下发到设备

将app升级到7.95+以支持新的国标平台

测试投递是否有外景照片
