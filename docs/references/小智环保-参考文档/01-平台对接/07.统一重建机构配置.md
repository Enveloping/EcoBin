统一重建机构配置

# 接口说明

接口作用，重新创建机构配置，当有变化后，机构配置的版本号会更新， 同时向机构下所有设备发送重新生成配置命令，设备的配置版本号也会更新，并通知设备更新配置。 如果设备离线，上线后继续同步新的配置。

接口地址：

    POST https://org.zsgai.com/open_api/updata_config

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

| 字段  | 数据类型 | 非空 | 默认值 | 说明     |
|:------|:---------|:-----|:-------|:---------|
| orgId | Long     | 否   |        | 机构编号 |

平台返回：

``` json
{
  "code": 0 // 0成功 其它失败
  "msg": "修改成功",
  "...": "忽略"
}
```
