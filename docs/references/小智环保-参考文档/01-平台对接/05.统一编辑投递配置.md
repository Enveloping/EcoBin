编辑机构投递配置文档

# 接口说明

编辑机构投递配置后，需要生成新的配置后(/org_config_updata)才会下发到设备,只有下发到设备后才会生效

接口地址：

    POST https://org.zsgai.com/open_api/updata_delivery_config

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

以下字段，用到哪个就携带哪个，为避免误操作不要全部携带，除非对设备的工作原理非常熟悉，当参数不生效或出现错误时，及时上传设备中的日志，看与设置的是否一至，检验是否成功下发。

| 字段 | 数据类型 | 非空 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| orgId | Long | 否 |  | 机构编号 |
| typeVerify | Integer |  | 1 | 投递审核方式(0:手动 1:自动 2:24H自动 3:48H自动 ){6} |
| typeValue | Integer | 否 | 0 | 取值方式(0:按投递重量 1:按投递数量 2:按投递次数 3:按登录次数) |
| scoreRule | Integer |  | 0 | 积分兑换规则\[1公斤可兑换多少积分\] |
| exchangePrice | BigDecimal |  | 0.00 | 兑换价\[1公斤可兑换多少元\] |
| dayWeight | BigDecimal |  | 0.00 | 当天上重量 |
| weekWeight | BigDecimal |  | 0.00 | 周上限重量 |
| monthWeight | BigDecimal |  | 0.00 | 月上限重量 |
| yearWeight | BigDecimal |  | 0.00 | 年上限重量 |
| dayMoney | BigDecimal |  | 0.00 | 当天上金额 |
| weekMoney | BigDecimal |  | 0.00 | 周上限金额 |
| monthMoney | BigDecimal |  | 0.00 | 月上限金额 |
| yearMoney | BigDecimal |  | 0.00 | 年上限金额 |
| minMoney | BigDecimal |  | 0.00 | 提现最小金额 |
| maxMoney | BigDecimal | 否 | 0.00 | 提现最大金额 |
| openMessage | String |  |  | 开门提示语 |
| closeMessage | String |  |  | 关门提示语 |
| dataFrom | Integer |  | 0 | 投递数据来源(0:设备累加 1:后台累加 2:设备即时重量){6} |
| promOpen | Integer |  | 0 | 开启推广(0:关闭 1:开启) |
| promExpTime | Integer |  |  | 有效期 |
| promMoney | BigDecimal |  | 0.00 | 推广金额 |
| promRegMoney | BigDecimal |  | 0.00 | 注册会员奖励金 |
| priceIn1 | Double |  |  | 1号投口单价 |
| priceIn2 | Double |  |  | 2号投口单价 |
| priceIn3 | Double |  |  | 3号投口单价 |
| priceIn4 | Double |  |  | 4号投口单价 |
| priceIn5 | Double |  |  | 5号投口单价 |
| priceIn6 | Double |  |  | 6号投口单价 |
| priceIn7 | Double |  |  | 7号投口单价 |
| priceIn8 | Double |  |  | 8号投口单价 |
| priceIn9 | Double |  |  | 9号投口单价 |
| priceIn10 | Double |  |  | 10号投口单价 |
| priceIn11 | Double |  |  | 11号投口单价 |
| priceIn12 | Double |  |  | 12号投口单价 |

平台返回：

``` json
{
  "code": 0 // 0成功 其它失败
  "msg": "修改成功",
  "...": "忽略"
}
```
