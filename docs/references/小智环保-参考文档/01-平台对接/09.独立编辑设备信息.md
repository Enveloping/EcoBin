编辑设备信息

# 接口说明

对指定的设备信息进行编辑

此文档所有的API修改后，需要生成新的配置后才会下发到设备后才会生效

接口地址：

    POST https://device.zsgai.com/open_api/edit_device_inf

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

以下字段，用到哪个就携带哪个，为避免误操作不要全部携带，除非对设备的工作原理非常熟悉，当参数不生效或出现错误时，及时上传设备中的日志，看与设置的是否一至，检验是否成功下发。

| 字段 | 数据类型 | 非空 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| devSn | String | 必填 |  | 设备编号 |
| name | String | 否 |  | 设备名称 |
| onLink | Integer |  | 0 | 在线状态(0:离线 1:在线){6} |
| weightsNow | Double |  |  | 即时重量 |
| weightsSys | Double |  |  | 理论重量 |
| spillNow | Integer |  | 0 | 即时满溢报警 |
| spillSys | Integer |  |  | 理论满溢报警 |
| spillAll | Integer |  |  | 全报则报 |
| spillTime | Date |  |  | 满溢时间 |
| smoke | Integer |  | 0 | 烟雾报警 |
| maintenance | Integer | 否 | 0 | 维护(0:正常 1:维护中){6} |
| progress | Integer |  |  | 更新进度 |
| offLinkConfig | Integer |  |  | 配置变更(0:无变更 1:未下发) |
| bagNo | String |  |  | 垃圾袋编码 |
| cleanTime | Date |  |  | 清运时间 |
| voltage | Double |  |  | 电压 |
| sim | String |  |  | 流量卡 |
| lat | Double |  |  | 纬度 |
| lng | Double |  |  | 经度 |
| promotion | Integer |  | 0 | 推广中(0:未推广 1:推广中){6} |
| promotionExpTime | Date |  |  | 推广过期时间 |
| cameraModel | Integer | 否 | 0 | 设像机厂商(0:未知 1:海康 2:莹石){6} |
| compress | Integer | 否 | 0 | 压缩状态(0:未压缩 1:压缩中 2:停止压缩 3:达到压缩上限){6} |
| compressProgress | Integer | 否 | 0 | 压缩进度 |
| deliveryTime | Date |  | CURRENT_TIMESTAMP | 最后投递时间 |
| doorPhoto | String |  |  | 设备满溢桶内照片 |

平台返回：

``` json
{
  "code": 0 // 0成功 其它失败
  "msg": "修改成功",
  "...": "忽略"
}
```
