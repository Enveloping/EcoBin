独立编辑投口配置

# 接口说明

编辑指定的投口配置， 此文档所有的API修改后，需要生成新的配置后才会下发到设备后才会生效

接口地址：

    POST https://device.zsgai.com/open_api/edit_door_config

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

以下字段，用到哪个就携带哪个，为避免误操作不要全部携带，除非对设备的工作原理非常熟悉，当参数不生效或出现错误时，及时上传设备中的日志，看与设置的是否一至，检验是否成功下发。

| 字段 | 数据类型 | 非空 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| doorId | Integer | 必填 |  | 门号( 0:无 1:1号 2:2号 3:3号 4:4号 5:5号 6:6号) |
| devSn | String | 必填 |  | 设备编号 |
| name | String |  |  | 垃圾分类别名 |
| type1 | Integer |  | 2 | 垃圾投口分类( 0:无 1:易腐垃圾 2:可回收垃圾 3:有害垃圾 4:其它垃圾) |
| type2 | Integer |  | 0 | 垃圾二级分类(0:不区分 1:金属 2:塑料 3:织物 4:纸类 5:其它) |
| checked | Integer |  | 0 | 是否选中(0:否 1:是)\[设置图片用\] |
| cleanLock | Integer |  | 0 | 是否有清运锁(0:否 1:是)\[清运时使用\] |
| smokeOn | Integer |  | 0 | 是否烟雾报警(0:否 1:是) |
| spillDistance | Integer |  |  | 满溢距离 |
| handDistance | Integer |  |  | 防夹手距离 |
| used | Integer |  | 0 | 是否占用(0:否 1是) |
| sort | Integer |  |  | 实显排序( 0:不显示 1:1号 2:2号 3:3号 4:4号 5:5号 6:6号) |
| show | Integer |  | 1 | 是否显示(0:关闭 1:显示) |
| scoreRule | Integer |  | 0 | 积分兑换规则\[1公斤可兑换多少积分\] |
| expiryTime | Integer |  |  | 有效期 |
| temper | Double |  | 0 | 温度 |
| weight | Double |  | 0 | 上一次重量 |
| weightsNow | Double |  | 0 | 即时重量 |
| weightsSys | Double |  | 0 | 理论重量 |
| spillNow | Integer |  | 1 | 即时满溢报警 |
| spillTime | Date |  |  | 满溢时间 |
| spillSys | Integer |  |  | 理论满溢报警 |
| smoke | Double |  | 1 | 烟雾(0:禁用 1:启用) |
| humidity | Double |  |  | 湿度 |
| spillOn | Integer |  | 1 | 是否满溢报警(0:否 1:是) |
| compress | Integer | 否 | 0 | 压缩状态(0:未压缩 1:压缩中 2:停止压缩 3:达到压缩上限) |
| compressProgress | Integer | 否 | 0 | 压缩进度 |
| compressNum | Integer |  | 0 | 压缩次数 |
| compressTimes | Integer |  | 0 | 压缩停留时间 |
| bagNo | String |  |  | 垃圾袋编号 |
| deliveryStatus | Integer |  |  | 投递门状态 |
| cleanStatus | Integer |  |  | 清运门状态 |
| price | BigDecimal |  |  | 投递单价 |
| typeValue | Integer |  | 0 | 计算方式(0:按重量 1:按数量) |
| typePrice | Integer |  |  | 结算方式(0:统一价) |
| merge | Integer |  | 0 | 多投口合一(0:不合并 1:合并) |
| enable | Integer |  | 0 | 是否启用(0:启用 1:禁用) |
| typeIco | String |  |  | 垃圾桶类别图片 |
| deliveryOrderSn | String |  |  | 投递订单 |
| cleanOrderSn | String |  |  | 清运订单 |
| bagNoNew | String |  |  | 新包裹码 |
| voltage | Double |  |  | 电压 |
| redirectDoorId | Integer | 否 | 1 | 合并到投口号 |
| maxWeight | Double | 否 | 0 | 报警重量 |
| cleanTime | Date |  |  | 清运时间 |

平台返回：

``` json
{
  "code": 0 // 0成功 其它失败
  "msg": "修改成功",
  "...": "忽略"
}
```
