# 抓拍图片(海康国标平台)

此接口仅适用于使用平台的提供的相机拍照，后端收到设备请求时，调用此接口进行相机抓拍，其他渠道的相机，请参照相关的相机厂商文档。

    如果有多种品牌的相机，可在设备配置中，修改`相机厂商`字段为国标相机，设备根据设置的值请求国标路径，

重要：请求到照片后，平台只缓存1小时，后端自行下载存储

请求路径：

    POST https://camera.zsgai.com/open_api/camera_capture_hk    # 仅适用于平台提供的海康国标相机 

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

| 序号 | 字段 | 类型 | 必报 | 示例 | 说明 |
|:---|:---|:---|:---|:---|:---|
| 1 | devSn | String | 是 | aa1ecdc26064b280 | 设备SN编码 |
| 2 | doorId | String | 是 | 1 | 投口编号 |
| 3 | type | String | 是 | 2 | 场景1内景，箱内照片2外景，箱外照片对应设备请求中的`type3` |
| 4 | sn | String | 是 | aa1ecdc26064b2807388256999960077 | 订单编码，当异步返回路径时，可与后端的订单绑定,原样返回 |

成功示例：

``` json
{
  "code":0,
  "picUrl":"https://hk.zsgai.com:6113/pic?1d00=cc02le5-do7718*56ecd602-0eb8a95b8*094s=**814==*p311=1t2679833000=0l2*2014=0od*446-122oe5-pi03a9=00620&AccessKeyId=/cNOGt3g4+k4L9d0&Expires=1721482438&Signature=Zx7wt2s5mHovscSezK6EDUlcmaU=",
  "devSn":"aa1ecdc26064b280",
  "doorId":"0",
  "type":"0",
  "sn":"aa1ecdc26064b2807388256999960077",
  "msg":"抓拍成功"
}
```

失败示例：

``` json
{
  "code":-1,
  "devSn":"aa1ecdc26064b280",
  "doorId":"0",
  "type":"0",
  "sn":"aa1ecdc26064b2807388256999960077",
  "msg":"抓拍失败"
}
```

后端返回值说明：

| 序号 | 字段     | 使用场景                                          |
|:-----|:---------|:--------------------------------------------------|
| 1    | `code`   | 0 成功 -1 失败                                    |
| 2    | `msg`    | 错误提示消息                                      |
| 3    | `picUrl` | 本次照片 请在1小时内下载，平台缓存1小时后自动删除 |
| 3    | `sn`     | 订单编号，原样返回                                |
