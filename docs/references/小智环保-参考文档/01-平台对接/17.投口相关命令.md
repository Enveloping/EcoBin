# 投口命令（危险！！！）

此命令仅用于管理员后台控制，不可开放给用户使用，以下命令都是强制操作，使用时误必要谨慎操作，确保箱体内无人，且无人操作设备，避免发生安全事故。

以下命令都以管理员userId=1的身份下发

再次强调：不可开放给用户使用！！

请求路径：

    POST https://device.zsgai.com/open_api/door_command

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、body:

| 字段     | 数据类型 | 非空 | 默认值 | 说明        |
|:---------|:---------|:-----|:-------|:------------|
| devSn    | String   | 必填 |        | 设备编号    |
| protocol | Integer  | 必填 |        | 固定值：171 |
| doorId   | Integer  | 必填 |        | 投口编号    |
| cmd      | String   | 必填 |        | 命令标识    |

命令标识

| 命令标识 | 说明 |
|:---|:---|
| clean-spill | 清除满溢，全部置零，屏蔽满溢传感器，重启后恢复默认(启用)状态，当传感器失灵时可临时使用 |
| compress-open | 开始压缩 非常危险，确保箱体内无人 |
| compress-close | 停止压缩 |
| delivery-open | 投递开门 |
| delivery-close | 投递关门 |
| declean-open | 清运开门 |
| weight-zero | 投口电子秤置零 |
| weight-500 | 设置投口电子秤校准后的重量为500克(固定值) |

返回：

| 序号 | 字段   | 使用场景       |
|:-----|:-------|:---------------|
| 1    | `code` | 0 成功 -1 失败 |
| 2    | `msg`  | 错误提示消息   |
