重建设备配置文档

# 接口说明

重新创建新的配置并下发到设备，只对传入的设备生效

接口地址：

    POST https://device.zsgai.com/open_api/create_device_config

请求参数：

1、 header：

| 序号 | 字段 | 使用场景 |
|:---|:---|:---|
| 1 | `Jnx-Access-Token` | 客户端请求时将`id_token`作为`Jnx-Access-Token`传递到服务端 |
| 2 | `Content-Type` | application/json |

2、 body：

以下字段，用到哪个就携带哪个，为避免误操作不要全部携带，除非对设备的工作原理非常熟悉，当参数不生效或出现错误时，及时上传设备中的日志，看与设置的是否一至，检验是否成功下发。

| 字段 | 数据类型 | 非空 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|
| orgId | Long | 必填 |  | 机构编号 |
| typeMerge | Integer |  | 0 | 同分类可投递(0:否 1:是) |
| maxWeight | Double |  | 100.00 | 报警重量(公斤) |
| delayCloseTime | Integer |  | 0 | 关门延时时间(秒) |
| logoutTime | Integer |  | 0 | 退出延时时间(秒) |
| guest | Integer |  | 0 | 允许游客模式(0:否 1:是) |
| merge | Integer |  | 0 | 多投口合一(0:否 1:是) |
| spillOn | Integer |  | 1 | 满溢(0:禁用 1:启用) |
| smokeOn | Integer |  | 1 | 烟雾(0:禁用 1:启用) |
| cleanLock | Integer |  | 0 | 是否安装清运锁(0:否 1:是) |
| enablePhoneLogin | Integer | 否 | 0 | 允许手机号登录(0:禁用 1:启用) |
| enablePhoneClean | Integer | 否 | 0 | 允许手机号清运(0:禁用 1:启用) |
| adsUrl | String |  |  | 设备首页的广告源地址 多个地址用英文逗号(`,`)分割 |
| thirdServerIp | String |  |  | 第三方服务器ip:port |
| thirdServerUrl | String |  |  | 第三方服务器接口 |
| thirdUserUrl | String |  |  | 第三方用户信息获取地址 |
| thirdCameraUrl | String |  |  | 第三方相机拍照地址 |
| thirdPhotoUrl | String |  |  | 第三方照片上传地址 |
| thirdFileUrl | String |  |  | 第三方文件上传地址 |
| thirdLogUrl | String |  |  | 第三方日志上传地址 |
| thirdDataUrl | String |  |  | 第三方数据接收地址 |
| thirdConfigUrl | String |  |  | 第三方配置路径 |
| thirdStatusUrl | String |  |  | 第三方设备状态推送地址 |
| thirdQrUrlPrefix | String |  |  | 第三方二维码前缀 用于生成自定义二维码跳转路径 |
| showCleanWeight | Integer |  | 1 | 设备首页显示清运重量(0:否 1:是) |
| showLastCleanTime | Integer |  | 1 | 设备首页显示最后清运时间(0:否 1:是) |
| compressNum | Integer |  | 6 | 压缩次数 控制设备的最大压缩次数，清运后归零 |
| compressTimes | Integer |  | 0 | 压缩停留时间(秒) 控制压缩机压缩到低部时的最大停留时间，通常需要多停顿一会(如5秒)才能压实 |
| uniqueDeliveryCode | Integer |  | 0 | 投递码唯一校验 |
| statusTime | Long |  | 0 | 状态上报间隔时间(秒) |

平台返回：

``` json
{
  "code": 0 // 0成功 其它失败
  "msg": "修改成功",
  "...": "忽略"
}
```
