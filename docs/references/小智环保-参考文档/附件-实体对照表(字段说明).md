# 公共参数

说明：所有设备App 接口都会上报相关参数，请按需处理。

| 序号 | 字段 | 类型 | 必报 | 默认值 | 说明 |
|:---|:---|:---|:---|:---|:---|
| 1 | devId | String | 是 |  | \`设备id 默认为1代表没有配置文件 |
| 2 | orgId | String | 是 |  | 机构id 同orgId 默认为1代表没有配置文件 |
| 3 | devSn | String | 是 |  | 设备SN编码 |
| 4 | devType | String | 是 |  | 设备类型 固定值 33 = 0x21 |
| 5 | appid | String | 是 |  | 小程序appid 默认为小智环保，需要在机构中修改 |
| 6 | clientType | String | 是 | "android" | 设备标识 固定值 android |
| 7 | clientVersion | String | 否 | 1 | 客户端app版本 |
| 8 | loginType | String | 否 |  | 登录类型 见附件后面的 \[loginType枚举\] |
| 9 | ltn | String | 否 |  | 登录类型描述 手机号登录 网络登录 （IC卡登录，游客登录） |
| 10 | test | String | 否 |  | 测试模式 1为测试模式 |
| 11 | ac | String | 否 |  | ac版本1 |
| 12 | appConfigVersion | String | 否 |  | 配置版本，在生成后下发到设备，观察屏幕上C*判断是否下发成功，*为对应的配置版本号，可在设备详情页面查看最新配置版本 |
| 13 | acc | String | 否 |  | ac版本2 |

# baseUrl 说明

请登录平台阅读字段的最新说明

| 序号 | 第三方 | 字段 | 说明 |
|:---|----|----|:---|
| 1 | 第三方tcp通讯 | thirdServerIp | 第三方完全控制设备 |
| 2 | 第三方api接口 | thirdServerUrl | 有值时开启 |
| 3 | 第三方状态推送 | thirdStatusUrl | 有值或启用第三方api接口 ：推送，无值不推送 |
| 4 | 第三方相机接口 | thirdCameraUrl | 开启第三方后且无值时使用第三方api接口 |
| 5 | 第三方照片接口 | thirdPhotoUrl | 开启第三方后且无值时使用第三方api接口 |
| 6 | 第三方文件接口 | thirdFileUrl | 开启第三方后且无值时使用第三方api接口 |
| 7 | 第三方日志接口 | thirdLogUrl | 开启第三方后且无值时使用第三方api接口 |
| 8 | 第三方数据接收接口 | thirdDataUrl | 开启第三方后且无值时使用第三方api接口 |
| 9 | 第三方二维码前缀接口 | thirdQrUrlPrefix | 开启第三方后且无值时使用第三方api接口 |
| 10 | 第三方配置接口 | thirdConfigUrl | 无值不使用，有值且开启第三方推送使用全路径 |
| 11 | 第三方用户接口 | USER_SERVER_URL | 开启第三方后且无值时使用第三方api接口 |
