# 二维码规则和路径配置指引

## 有独立的服务器（小程序指向自己的服务器）

前往小程序配置`扫普通链接二维码打开小程序`，配置成功后，将此路径填写到平台的设备配置中，并下发到设备

## 没有服务器，依赖平台的小程序：

登录小程序后台启用`扫普通链接二维码打开小程序`

- 添加二维码规则: `https://dev-api.zsgai.com/device/api/dev_qr_code/:appid/`

- 下载校验文件

- 联系我们技术人员添加校验文件进行校验，

校验成功后，自行添加到平台，并下发到设备

## 跳转地址如何添加到平台

全局配置 可添加到机构的配置中，机构下的所有设备生效，

独立配置 可添加到设备的配置中，设备独立生效，

添加到设备中的优先使用
