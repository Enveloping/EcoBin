# 如何导出原始记录

客户希望导出数据进行更多的维度分析，在列表页面只能导出30条，希望导出更多的数据，在哪里可以导出呢？

操作步骤：

1、 打开机构列表页面

    https://xzhb.zsgai.com/org/list

2、点击对应机构列表中的【导出数据】打开导出页面

3、在导出页面中选择需要导出的数据，

可选择不同的月份，支持投递、清运、提现、充值的数据导出，

# 导出统计数据

打开对应的统计页面，点击【将查询结果导出到Excel文件】

由于需要的算力较高，后台为即时统计，为避免宏机，因此统计数据不支持批量导出，

但可支持每页50条的结果导出，超出50条记录的客户需要多次翻页导出，

设备列表支持每页200条的导出，超出200台的客户需要多次翻页导出。

以导出设备统计数据为例：

1、打开设备统计页面 选择对应的月份

        https://stat.zsgai.com/month_clean_device 

2、在列表低部选择每页导出200行，

3、点击【将查询结果导出到Excel文件】

4、在浏览器下载中找到刚才导出的文件。
