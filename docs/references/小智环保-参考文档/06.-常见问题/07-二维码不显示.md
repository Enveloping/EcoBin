# 二维码不显示

### 故障现象：

1、以前已配置好的配置，断电重启后二维码不显示，提示无配置，设备id和机构id都为1

2、全新的设备也会没有配置

### 故障原因：

断电后，网络没有恢复的同时(网络设备启动的速度较慢)，且本地数据有可能损坏，app此时使用了默认的配置启动。

### 解决办法：

##### 现象1：

1、删除损坏的db文件，/sdcard/jnx-db 目录内的文件全删除

2、重启app,（或者从后台重启设备）

3、从后台再次下发配置到设备(设备页面 -\> 生成配置)

三个步骤顺序不能错

##### 现象2：

1、从后台下发配置到设备(设备页面 -\> 生成配置)，下发成功后设备会自动刷新界面。

下发成功后设备会自动刷新界面，如果失败，确保网络正常，当信号较弱时可重试几次。
