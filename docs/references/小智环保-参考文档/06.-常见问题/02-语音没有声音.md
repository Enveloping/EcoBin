# 1.语音没有声音

问题现象：音量登录的时候不提示登录成功、结束的时候投递收益提示语音，或者是开门、关门语音

排查路线：

## 验证喇叭是否有声音

1、设备音量调整到最大

2、连网测试硬件：当设备有网络时，检查网页中的视频是否有声音：

3、无法联网时：调整系统音量大小，看能否听到系统的提示音，或本地通过U盘放入语音文件或视频文件，用内置的播放器打开，验证喇叭是否有声音。

## 验证是否有语音授权

当上面的步骤无效时，再排查此步骤

上传设备日志，找查`合成引擎初始化成功` 字样，如果存在就已授权了，则仍需要排查硬件问题。
