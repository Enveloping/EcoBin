# 1.外景相机不拍照

问题现象：外部监控实时可以查看，但是投递记录没有外部抓拍

当设备有网络时，检查下面的可能出现故障的原因：

1、设备未选择使用的相机厂商

2、配置投口相机时，未指定为外景相机

3、配置投口相机时，外景的投口编号不为 0

4、没有网络：检查日志中的请求信息，是否为网络不可用。

# 2.内景相机不拍照

可能出现故障的原因：

1、测试本地是否可以正常拍照，

操作方法：退出app,用设备的内置相机软件预览摄像头，检查是否可以正常工作,如果不能工作，需要更换摄像头

2、检查usb相机的连接线

3、没有网络：检查日志中的请求信息，是否为网络不可用。

4、如果上述步骤都正常，即可投递测试
